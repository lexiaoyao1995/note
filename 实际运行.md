# Spring

### @PostConstruct

<https://www.jianshu.com/p/98cf7d8b9ec3>

~~~java
    public Class AAA{   
        @Autowired   
        private BBB b;    
        
        public AAA() {       
            System.out.println("此时b还未被注入: b = " + b);   
        }
           
        @PostConstruct   
        private void init () {       
            System.out.println("@PostConstruct将在依赖注入完成后被自动调用: b = " + b);
        }
    }
~~~
